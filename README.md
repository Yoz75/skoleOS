# What is it
SkoliOS is a small UEFI application [with a crooked spine]. I'm not a pro in osdev, this project is made to proof that it is possible to make an UEFI application **using dub on Windows**.

# Dependencies
* Windows (actually it is possible to modify the dub.json file and run it on Linux)
* dub
* uefi-d
* Qemu
* OVMF (already included!!!)

# Building
* runqemu.bat -- run in debug mode
* runqemur.bat -- run in release mode
